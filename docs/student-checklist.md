# Student Checklist

Before presenting or submitting this project, make sure you can explain:

- what each bit in a portfolio bitstring means;
- why expected return is good and risk is penalized;
- what the covariance matrix represents;
- why a QUBO can be mapped to an Ising Hamiltonian;
- what the QAOA `gamma` and `beta` parameters control;
- why a classical optimizer is still needed;
- how to interpret the probability bar chart.

If you can explain these points in your own words, you understand the main logic of the notebook.
