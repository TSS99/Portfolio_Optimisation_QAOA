# Report Outline

Students can use this outline for a short project report.

1. Introduction
   Explain the portfolio optimization problem and why QAOA is being used.
2. Mathematical model
   Describe the binary variables, expected returns, covariance matrix, and risk factor.
3. QUBO to Ising mapping
   Explain how the binary objective becomes a quantum cost Hamiltonian.
4. QAOA circuit
   Describe the cost layers, mixer layers, and trainable parameters.
5. Optimization process
   Explain how the classical optimizer updates the parameters.
6. Results
   Interpret the most likely bitstrings and the probability chart.
7. Limitations and future work
   Mention the small mock dataset, simulator use, and possible extensions.

Keep the explanation focused on the chain of logic rather than only showing code screenshots.
