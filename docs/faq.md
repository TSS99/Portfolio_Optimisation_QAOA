# FAQ

## Is this financial advice?

No. The project is for learning quantum optimization concepts.

## Why are there only three assets?

Three assets keep the state space small. There are only eight possible portfolios, so students can inspect the result easily.

## Why use QAOA for this?

QAOA is a common quantum optimization algorithm. Portfolio optimization is a useful example because it naturally becomes a binary decision problem.

## Why is a simulator used?

A simulator makes the notebook easier to run and debug. Real quantum hardware introduces noise, queue times, and backend-specific details.
