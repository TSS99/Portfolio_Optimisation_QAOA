# Prerequisites

This project is beginner-friendly, but a few basics make it much easier to follow.

Helpful background:

- Python lists, arrays, and functions;
- basic matrix multiplication;
- simple probability ideas;
- what a qubit is;
- the difference between classical and quantum algorithms;
- basic Jupyter Notebook usage.

You do not need advanced finance or advanced quantum mechanics to start. The notebook introduces the required ideas step by step.
