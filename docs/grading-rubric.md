# Suggested Grading Rubric

This rubric can help evaluate a student presentation or report.

| Area | What to look for |
| --- | --- |
| Problem understanding | Clearly explains portfolio optimization as a binary decision problem |
| Finance logic | Correctly describes expected return, risk, covariance, and `q` |
| Quantum mapping | Explains why QUBO is converted to an Ising Hamiltonian |
| QAOA workflow | Describes cost layers, mixer layers, and parameter optimization |
| Results | Correctly interprets bitstrings and probability bars |
| Limitations | Mentions simulator use, small asset count, and heuristic behavior |
| Communication | Uses beginner-friendly language and a logical sequence |

The strongest submissions explain not only what the code does, but why each step is needed.
