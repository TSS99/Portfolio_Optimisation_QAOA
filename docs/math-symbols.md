# Math Symbols

This note summarizes the main symbols used in the notebook.

| Symbol | Meaning |
| --- | --- |
| `n` | Number of assets |
| `x` | Binary portfolio decision vector |
| `x_i` | Decision for asset `i` |
| `mu` | Expected returns vector |
| `Sigma` | Covariance matrix |
| `q` | Risk-aversion factor |
| `H_C` | Cost Hamiltonian |
| `H_M` | Mixer Hamiltonian |
| `p` | Number of QAOA layers |
| `gamma` | Cost-layer angle |
| `beta` | Mixer-layer angle |

When reading the notebook, first identify which symbol belongs to finance, which belongs to the optimization model, and which belongs to the quantum circuit.
