# Glossary

| Term | Beginner-friendly meaning |
| --- | --- |
| Asset | Something that can be included in a portfolio, such as a stock or fund |
| Portfolio | A selected group of assets |
| Expected return | The average return we hope an asset will produce |
| Risk | Uncertainty in the return |
| Covariance | A measure of how two assets move together |
| QUBO | An optimization problem with binary variables and quadratic terms |
| Ising Hamiltonian | A quantum-friendly way to represent the same cost function |
| QAOA | A hybrid quantum-classical algorithm for optimization |
| Mixer | The part of QAOA that helps explore different bitstrings |
| Expectation value | The average cost predicted by the quantum state |
